# Sample data description

The sample CSV file demonstrates the input format expected by the demo scripts. It is a small anonymized/synthetic-format sample and is not the complete field dataset used in the manuscript.

Columns:

- `HV`: harvester velocity
- `TCS`: threshing cylinder speed
- `FS`: fan speed
- `BR`: kernel breakage rate
- `IR`: impurity rate
- `LR`: loss rate

The full field dataset is not included because of confidentiality restrictions related to field trials and agricultural machinery operation.
